Fixed Grid Sizing

So how do we actually define a grid? It is pretty easy.

display: grid;
width: 500px;
height: 500px;
border: 1px solid black;
grid-template-columns: 50px 120px 110px 120px 100px;
grid-template-rows: 220px 150px 130px;
120px / 220px
110px / 220px
120px / 130px
